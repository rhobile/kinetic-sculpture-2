# **App Name**: Kinetic Sculptures

## Core Features:

- Video Thumbnails: Thumbnails for videos uploaded to cloud storage.
- Video Upload: Videos are uploaded to cloud storage.
- Masonry Layout: Thumbnails are displayed on a masonry 4-column layout to the right. Clicking a thumbnail starts the associated video.
- Sidebar Navigation: A simple left-hand sidebar with introductory text and links to 'about' and 'contact'.

## Style Guidelines:

- Background color: white
- Body and headline font: 'helvetica' suitable for headlines or body text
- Highlighted text link to aid navigation
- Clean, grid-based layout with no borders and minimal padding.
